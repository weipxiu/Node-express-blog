## 如何运行
在mongodb安装目录下去执行以下命令，映射到自己项目的db目录中
>mongod --dbpath=E:\personal\Node-express-blog\db --port=27017

> node 版本 `>=8.11.3`

```bash
# 安装依赖
npm install

# 本地运行
npm run dev

# 本地运行地址
localhost:8080

```
